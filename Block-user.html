Empêcher un utilisateur d'accéder à vos backends avec haproxy

ven. 14 mars 2014
By cyrilb

Empêcher un utilisateur d'accéder à votre site internet qui est derrière haproxy est très simple, et autant le bloquer directement depuis haproxy.

Une fois encore, la souplesse des acl de haproxy permet de le faire en 30 secondes.

Exemple pour bannir une adresse ip :

acl bad_ip src www.xxx.yyy.zzz
tcp-request connection reject if bad_ip

On peut bien sûr remplacer l'adresse ip par une liste d'ip stockées dans un fichier plat :

acl bad_ip src -f /opt/blacklist.txt
tcp-request connection reject if bad_ip
